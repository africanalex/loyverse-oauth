Loyverse OAuth

OAuth integration library for Loyverse API

About

Loyverse OAuth is a library designed to simplify OAuth authentication with the Loyverse API. Loyverse is a comprehensive point-of-sale (POS) and inventory management system that provides APIs for developers to integrate with their business applications.

This project provides tools and utilities to handle OAuth 2.0 authentication flow with Loyverse's API, making it easier for developers to build applications that interact with Loyverse merchant data securely.

Features

🔐 Secure Authentication

Handle OAuth 2.0 flow with proper security measures and token management.

📚 Easy Integration

Simple API design that makes integration with Loyverse services straightforward.

🔄 Token Management

Automatic token refresh and secure storage capabilities.

Quick Start

Get started with Loyverse OAuth integration:

Installation

npm install loyverse-oauth

Basic Usage

import { LoyverseOAuth } from 'loyverse-oauth'; const oauth = new LoyverseOAuth({ clientId: 'your-client-id', clientSecret: 'your-client-secret', redirectUri: 'your-redirect-uri' }); // Start OAuth flow const authUrl = oauth.getAuthorizationUrl(); window.location.href = authUrl;
